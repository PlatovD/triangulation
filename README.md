## Описание

Класс `EarCuttingTriangulator` реализует интерфейс `Triangulator` и предоставляет функциональность для разбиения полигонов на треугольники с использованием метода "отрезания ушей" (ear-cutting).

## Основные методы

### `triangulatePolygon(Model model, Polygon polygon)`

Основной метод для триангуляции полигона. Принимает на вход модель и полигон, возвращает список треугольников (полигонов с 3 вершинами).

**Логика работы:**
1. Если полигон имеет менее 4 вершин, возвращает его как есть (уже треугольник или линия)
2. Использует очередь вершин для последовательной обработки
3. Проверяет каждую вершину на "выпуклость" и является ли она "ухом"
4. Создает новые треугольники по мере обработки вершин

### Вспомогательные методы

#### `checkEar(int leftPointIndex, int rightPointIndex, int middlePointIndex, Map<Integer, Vector3f> vertices)`

Проверяет, является ли треугольник, образованный тремя точками, "ухом" полигона. "Ухо" - это треугольник, который не содержит внутри себя других вершин полигона.

#### `leftTurn(Vector3f a, Vector3f b, Vector3f c)`

Вычисляет векторное произведение для определения ориентации трех точек. Возвращает:
- Положительное значение: левый поворот
- Отрицательное значение: правый поворот
- Ноль: точки коллинеарны

## Особенности реализации

- Алгоритм работает с полигонами любой формы (выпуклыми и вогнутыми)
- Использует очередь для циклического обхода вершин
- Проверяет каждую потенциальную "уху" на отсутствие внутренних вершин
- Создает треугольники последовательно, уменьшая полигон до полной триангуляции
